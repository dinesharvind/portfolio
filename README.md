# Personal Portfolio Website (v1)

This repository contains the source code for my personal portfolio website.
The site showcases my background, projects, and technical skills.

## Overview
- Desktop-first personal portfolio website
- Built to practice UI layout, styling, and basic JavaScript interactions
- Deployed using GitHub Pages

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Single-page layout with structured sections
- Basic animations and interactive elements using JavaScript
- Clean, minimal design focused on readability
